# 味觉密码学 | Gustatory Cryptography

本项目通过先进算法破译隐藏在海量食谱数据中的味觉偏好密码，为每个用户解锁专属的美食世界。

## 拉取 docker 镜像

```bash
docker pull quay.io/jupyter/pyspark-notebook
docker run -d -p 8888:8888 --name notebook quay.io/jupyter/pyspark-notebook:latest
```
